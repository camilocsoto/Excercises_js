¿cómo envíar información al servidor?

1. Copia el siguiente código en la consola del navegador

 
        
            socket.emit('enviarMensaje', { //nombre de la variable y el json para enviar info al server
                usuario: 'user',
                mensaje: `--escribe aquí para comunicarte con el servidor--`
            });

2. Dale enter, y vé a la vista del servidor, allí valida que llegó el mensaje correctamente
